The program utilizes the aima-core.jar library.
To run the program from the command line simply compile with the command

javac -cp aima-core.jar PuzzleSolver.java RecursiveBestFirstSearch.java

Run the program with

java -cp .;aima-core.jar PuzzleSolver (If using windows)
java -cp .:aima-core.jar PuzzleSolver (If using linux)

Included is a sample puzzle in the puzzle.txt file, use this file to edit or add a new puzzle.
